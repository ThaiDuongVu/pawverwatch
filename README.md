## Guide

How to use **<span style="color: #4a4c4eff">Paw</span><span style="color: #FA9C1D">verwatch</span>**

1. Go to [Pawverwatch.com](https://pawverwatch.com/)
2. Upload a base pet image
<img style="border:0.25rem solid grey; border-radius:0.5rem" src="/public/images/screenshots/3.png" width="400" alt="screenshot1"/>
3. (Optional) Remove the background from the image
<img style="border:0.25rem solid grey; border-radius:0.5rem" src="/public/images/screenshots/4.png" width="400" alt="screenshot2"/>
4. Click on a hero, then add the items you want
<img style="border:0.25rem solid grey; border-radius:0.5rem" src="/public/images/screenshots/5.png" width="400" alt="screenshot3"/>
5. Edit the added items
<img style="border:0.25rem solid grey; border-radius:0.5rem" src="/public/images/screenshots/6.png" width="400" alt="screenshot4"/>
